# RePERFORMANCE Homepage v1

Next.js App Router 기반의 RePERFORMANCE 랜딩페이지 v1입니다.

## 실행

```bash
npm install
npm run dev
```

로컬 주소: http://localhost:3000

## 배포

```bash
vercel --prod
```

또는 GitHub에 업로드 후 Vercel 프로젝트와 연결하세요.
